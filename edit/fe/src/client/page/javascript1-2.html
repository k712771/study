body보다 더 큰 영역이 있다?!

document를 HTML 구성으로 head 와 body로 나뉘어진다.
head와 body를 둘러싼 최상위 요소가 document.documentElement로 반환되는 것을 알수있다!
결론은 body 보다 더 큰 document.documentElement 의 영역이 존재한다는 것!

백분율의 계산법(암산 말고 암기)

( 작은 수 / 큰 수 ) * 100
※주의할 점
암산 하지 말고 계산법을 이용을 하려 하자

Math의 메소드는 무엇이 있을까?

Math.min() = 최솟값 반환
Math.max() = 최댓값 반환
Math.random() = 0 보다 크거나 같고 1보다 작거나 같은 랜덤의 수
Math.round() = 소수점 첫 번째 자리에서 반올림하여 그 결괏값을 반환
Math.floor() = 같거나 작은 수 중에서 가장 큰 정수를 반환
Math.ceil() = 같거나 큰 수 중에서 가장 작은 정수를 반환
Math.sin() = 사인(sine) 함숫값을 반환

오류 발견

백분율 계산 과정을 적는 도중에
Nan 과 infinity 오류가 났었다.
Nan은 숫자가 아닌 문자열로 구분이 되어 반환했다는 뜻이고, infinity는 무한대라는 뜻이다.
내가 적은 js코드를 다시 확인해보니 무조건 숫자로 반환해주기때문에 오류를 이해할수 없었다.
차근차근 읽어보니 오타가 문제였다.. ㅡㅡ;

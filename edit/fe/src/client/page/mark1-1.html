Grid layout 사용하기

마크업1-1 2023년04월10일

설명

복잡한 layout을 grid의 규칙을 파악하고 적절하게 사용할줄만 안다면 쉽게 구성해낼수 있다.
또한 현재 날짜 기준으로 익스플로러의 사망(?)으로 인해 grid를 사용하는데 있어 크게 제한이 없다.
flex와 병행을 한다면 복잡한 layout을 빠르게 구성할수 있고 수정도 쉬워지는것을 알수 있었기에
나는 적극적으로 grid 사용하는 것을 좋아하는 편이다.
mdn에서는 여러개의 용어들이 제시되어있지만 먼저 가로와 세로를 이용해 구성을 한다면 쉽게 이해가 가능하다.
grid-template-columns(가로)과 grid-template-rows(세로)로 디자인과 가깝게 구성을 만들어내면 된다.
